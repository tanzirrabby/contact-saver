Contact Saver App (React + Node.js)

1. Backend Setup:
-----------------
cd contact-saver/backend
npm install express cors body-parser
node server.js

2. Frontend Setup:
------------------
cd contact-saver/frontend
npx create-react-app .
npm install axios
Replace src/App.js with the provided App.js
npm start

Visit http://localhost:3000 (or other port it opens on)
